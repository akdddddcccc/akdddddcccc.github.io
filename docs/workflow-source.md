# AI直播间贴片生成项目原始工作流文档

把最初生图效果最好的那份 Markdown 文档放在这里，或在 `.env.local` 里用 `AI_WORKFLOW_DOC_PATH` 指向原始文件路径。

生成服务会在每次贴片背景和文字图层请求前读取这份文档，并把它作为最高优先级的制作准则提供给模型。

建议文档保留这些内容：

- 参考图如何拆解为直播间贴片语言。
- 上贴、侧贴、下贴各自的构图、渐隐和留白规则。
- 色彩、饱和度、材质、光效、装饰密度的判断标准。
- 禁止生成的元素，例如文字、logo、人物、商品、二维码、促销标签和海报排版。
- 文字图层如何继承上贴的颜色、材质和装饰细节。
- 最初测试中效果好的 prompt、反向约束和质量检查标准。
